# Project Memento: An Organizational Memory System for Software Development

See [IDEA.md](./IDEA.md) for the full concept.

*Memento oblivisci*: "Remember to forget"

a paradoxical phrase that captures the essence of our approach to organizational memory. In software development, we often struggle with information overload, where the sheer volume of data can lead to cognitive fatigue and decision paralysis. Project Memento is designed to help teams remember what they need to know while forgetting what they don't, creating a more efficient and effective way to manage knowledge.
